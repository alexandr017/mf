# Сидер для тестовых матчей

## Описание

Сидер `LiveMatchesSeeder` создает несколько тестовых матчей, готовых к запуску live трансляций через админ-панель.

## Использование

### Запуск сидера

```bash
php artisan db:seed --class=LiveMatchesSeeder
```

Или через DatabaseSeeder (если добавлен):

```bash
php artisan db:seed
```

### Что делает сидер

1. **Проверяет наличие команд** - нужно минимум 2 активные команды
2. **Находит последний сезон** - использует последний созданный сезон
3. **Создает или находит стадию** - создает тестовую стадию, если не существует
4. **Создает 5 матчей** с разными командами:
   - Статус: `scheduled` (готов к запуску)
   - Разные даты (каждый следующий матч на день позже)
   - Время: 15:00
   - Команды выбираются случайно, без дубликатов пар

### Требования

- Минимум 2 активные команды в базе
- Хотя бы один сезон турнира
- Команды должны иметь игроков в текущем сезоне (для генерации результата)

## Результат

После выполнения сидера вы получите:

- 5 тестовых матчей в статусе `scheduled`
- Все матчи готовы к запуску через админ-панель
- Каждый матч имеет две команды (team_1 и team_2)

## Запуск матчей

1. Перейдите в админ-панель: `/admin/matches`
2. Найдите созданные матчи
3. Нажмите кнопку **"Запустить"** для любого матча
4. Матч начнется и будет доступен по ссылке `/live-matches/{matchId}`

## Настройка

Вы можете изменить количество создаваемых матчей в файле `database/seeders/LiveMatchesSeeder.php`:

```php
$matchesToCreate = 5; // Измените на нужное количество
```
