# Быстрый старт: Live матчи

## Запуск матча через админку

1. **Войдите в админ-панель** → раздел **"Матчи"**
2. **Найдите матч** с указанными командами (team_1 и team_2)
3. **Нажмите "Запустить"** (зеленая кнопка)
4. **Нажмите "Смотреть"** для просмотра матча

## Просмотр матча

- **Ссылка**: `/live-matches/{matchId}`
- **Длительность**: 15 минут
- **Обновление**: автоматически каждую секунду

## Что видно на странице

- ✅ Счет матча в реальном времени
- ✅ Поле с игроками (Canvas)
- ✅ События матча (голы, передачи)
- ✅ Текущая минута

## Автоматическое завершение

Матч завершится через 15 минут, результат сохранится автоматически.

## Принудительное завершение

В списке матчей нажмите **"Завершить"** (желтая кнопка).

## Настройка планировщика

Для автоматической обработки добавьте в cron:
```bash
* * * * * cd /path-to-project && php artisan schedule:run >> /dev/null 2>&1
```

Или запускайте вручную:
```bash
php artisan live-matches:process
```
